üíæ Storage & Backup Information

Database Location:

C:\Users\jamal\OneDrive\Desktop\Hostel\db.sqlite3

Database Tables:

  • hostel_app_room - Stores all room information (capacity, price, amenities)
  • hostel_app_guest - Stores guest details (name, email, phone, address)
  • hostel_app_booking - Stores all bookings with dates, prices, and status
  • auth_user - Admin user accounts for login

How to Access:

  • Django Admin: /admin/ (Recommended)
  • SQLite Browser: Download DB Browser for SQLite and open db.sqlite3
  • Python Script: Write custom scripts to query the database

Backup Your Data:

  • Copy the file: db.sqlite3
  • Store it in a safe location
  • Create regular backups to avoid data loss
