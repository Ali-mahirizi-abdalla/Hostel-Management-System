# Overview

Hostel Management is a Django-based web application for small hostels and budget hotels to manage rooms, guests, and bookings.

Core capabilities:

- Manage room inventory (types, capacity, price, amenities, availability)
- Create and manage guest records
- Bookings with check-in/check-out tracking
- Protected data viewer for internal staff (login required)
- Responsive, modern UI with professional styling
- Admin panel available via `/admin/` for full control

Target users:

- Hostel managers who want a simple, user-friendly interface to manage reservations and view occupancy stats.

Design goals:

- Simple deployment on common hosting providers (Render, Docker/VPS)
- Clean, mobile-first UI
- Minimal external dependencies for easy maintenance
